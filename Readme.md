# 📄 PDF Insight AI: Professional RAG Application

An end-to-end Retrieval-Augmented Generation (RAG) system that allows users to chat with PDF documents in real-time. Built with a focus on modularity and scalability using Mistral AI, LangChain, and Streamlit.

## 🚀 Features
- **PDF Processing:** Automated text extraction and cleaning.
- **Advanced Retrieval:** Implements **MMR (Maximum Marginal Relevance)** to reduce redundancy in retrieved context.
- **Vector Storage:** Persistent vector database using **ChromaDB**.
- **Interactive UI:** Clean, chat-based interface built with **Streamlit**.
- **Model Agnostic:** Configured for Mistral AI, with easy integration for OpenAI or Hugging Face models.

## 🛠️ Tech Stack
- **Orchestration:** LangChain
- **LLM:** Mistral AI (mistral-small-2402)
- **Embeddings:** MistralAIEmbeddings
- **Vector Database:** ChromaDB
- **Frontend:** Streamlit
- **Language:** Python 3.x
